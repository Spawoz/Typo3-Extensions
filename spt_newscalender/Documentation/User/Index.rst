Users Manual
============
   #.Install the news extension.
   
   #.Install spt_newscalender extension.

   #.Include the plugin in root static template.

   #.Place the plugin in the page where the calender is to be displayed. Add the news detail page in the plugin flexform option.

   #.Add the news articles in news storage folder.

   #.spt_newscalender extension extends news extension for two options;
     
      Repeated News:Used to repeat the news within an interval of one week until end Date.

      Exclude From Event Calender:Used to exclude news from calender.
